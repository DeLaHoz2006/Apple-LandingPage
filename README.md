# Apple Landing Page

Static landing page inspired by Apple, built with HTML, CSS and JavaScript.

## Live Demo

Main page:
`https://delahoz2006.github.io/Apple-LandingPage/`

Secondary page:
`https://delahoz2006.github.io/Apple-LandingPage/fifty-years.html`

## Local Preview

Open `index.html` in your browser, or use a local server if you want automatic reload.

## Project Structure

- `index.html`: main landing page
- `fifty-years.html`: secondary promotional page
- `src/css`: project styles
- `src/js`: carousel logic and interactions
- `src/img`: image assets
- `src/video`: video assets
